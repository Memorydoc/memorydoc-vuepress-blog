## Guava事件机制


## Guava事件
```java 
public class GuavaEvent {
    @Subscribe
    public void subscribe(String string){
        // todo 业务逻辑
        System.out.println("订阅内容" + string);
    }
}

```

## 测试
```java 
    public static void main(String[] args) {
        EventBus eventBus = new EventBus();
        GuavaEvent guavaEvent = new GuavaEvent(); // 事件
        eventBus.register(guavaEvent); // 注册事件
        eventBus.post(" event parameter"); // 触发事件
    }
```
